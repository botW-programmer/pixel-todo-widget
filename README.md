Pixel-todo is a todo-list widget that sits right on your desktop and runs as a background project!
The background and "my quests" title are both replaceable. To edit them, simply save a new background.png to replace the old one (same goes for the title). Check the code files!
It saves completely locally.
I created this project as a fun little way to pass the time, it helps me stay organized and remember which coding projects I need to work on!
